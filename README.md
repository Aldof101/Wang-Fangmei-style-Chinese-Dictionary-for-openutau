Wang Fangmei-style Chinese Dictionary for openutau_ver1.0
by Aldof
This dictionary is based on the modified Wang Fangmei-style recording table (reference: https://groupbighand.weebly.com/wang-fang-mei-list.html). It enables Wang Fangmei-style Chinese voicebanks in OpenUtau to automatically split sounds by directly inputting Chinese characters or pinyin.
This YAML file references this repository: https://github.com/stakira/OpenUtau/pull/776

Usage:

1. Copy the "zhcvv.yaml" file from the extracted folder to the directory of the Wang Fangmei-style Chinese voicebank in OpenUtau. For example, paste "zhcvv.yaml" into E:\OpenUtau\Singers\FangmeiChinese.

2. In OpenUtau, select "ZH CVV (October-style Phoneme Expansion)" as the phonemizer.

3. Refresh the singer, and directly input Chinese lyrics or pinyin in the editing interface to automatically split sounds.

王芳梅式中文辞書 for openutau_ver1.0
by Aldof
この辞書は改良版の王芳梅式収音テーブルを基に作成されています（参考: https://groupbighand.weebly.com/wang-fang-mei-list.html）。王芳梅式の中文音源をOpenUtauで使用する際、直接中国語またはピンインを入力することで、自動的に音分割が行えます。
本yamlファイルは、以下のリポジトリを参考にしました：https://github.com/stakira/OpenUtau/pull/776

使用方法：

1、解凍後のフォルダ内にある「zhcvv.yaml」を、OpenUtauで王芳梅式中文スキームを使用する音源フォルダにコピーします。例：E:\OpenUtau\Singers\FangmeiChinese に「zhcvv.yaml」を貼り付けます。

2、OpenUtauで「ZH CVV（十月式整音拡張）」を音素器として選択します。

3、歌手を更新し、編集画面で直接中国語の歌詞またはピンインを入力すると、自動的に音分割が行われます。
