Wang Fangmei Style Chinese Dictionary for OpenUtau_ver1.1
by Aldof
This dictionary is based on the modified Wang Fangmei recording table (reference: https://groupbighand.weebly.com/wang-fang-mei-list.html). It allows Chinese vocal libraries following the Wang Fangmei style to automatically split lyrics in OpenUtau by directly inputting Chinese characters or Pinyin.
This YAML file references the following repository: https://github.com/stakira/OpenUtau/pull/776

Version 1.1: Updated the Modified Fangmei table (ModifiedFangMei) and the original Wang Fangmei table (OldFangMei), adding some missing pronunciations for the original table. Please select the appropriate version according to the vocal library's recording scheme.

Usage:

1. Copy the zhcvv.yaml file from the extracted folder and paste it into the folder of the Wang Fangmei style Chinese vocal library in OpenUtau. For example, paste zhcvv.yaml into E:\OpenUtau\Singers\FangmeiChinese.

2. In OpenUtau, select the ZH CVV (October-style Extended Phonemizer) phonemizer.

3. Refresh the singer, and you can directly input Chinese lyrics or Pinyin in the editor for automatic lyric splitting.

王芳梅式中国語辞書 for OpenUtau_ver1.1
by Aldof
この辞書は改良版王芳梅式録音表を基に作成されています（参考: https://groupbighand.weebly.com/wang-fang-mei-list.html）。王芳梅式中国語音源を使用し、OpenUtauで中国語またはピンインを直接入力することで、自動的に音節分割を行えます。
このYAMLファイルは以下のリポジトリを参考にしています: https://github.com/stakira/OpenUtau/pull/776

バージョン1.1: 改良版王芳梅表（ModifiedFangMei）と原版王芳梅表（OldFangMei）を更新し、原版王芳梅表の一部欠落していた発音を補完しました。音源の録音方式に合わせて適切なバージョンをご使用ください。

使用方法:

1、解凍したフォルダ内のzhcvv.yamlを、OpenUtauで王芳梅式中国語音源を使用するフォルダにコピーします。例: zhcvv.yamlをE:\OpenUtau\Singers\FangmeiChineseに貼り付けます。

2、OpenUtauでZH CVV（十月式整音拡張）フォネマイザーを選択します。

3、歌手を更新し、編集画面で中国語歌詞またはピンインを直接入力すると、自動的に音節分割が行われます。
